User Interfaces - Assignment 1 - Parking Kiosk

- Landscape orientation (1280x720)
	- to better fit keyboard

- Larger keys for big fingers 
	-larger font (easier to read/distinguish)

- Only allow digits in the student # / pin fields (numeric keyboard vs alphabet keyboard)
	-also limit # characters for each field (PIN only allows 4 digits/ S# only 9)

- Add drop down menu for expiry insurance company/expiry date

- Email Address optional
- Email Rules:
	Uppercase and lowercase English letters (a-z, A-Z)
	Digits 0 to 9
	Characters ! # $ % & ' * + - / = ? ^ _ ` { | } ~
	Character . (dot, period, full stop) provided that it is not the first or last character, and provided also that it does not appear two or more times consecutively.
